## awstk env set

環境変数の設定方法を表示

### Synopsis

指定した環境変数を設定するためのexportコマンドを表示します。

例:
  awstk env set -S my-stack
  awstk env set -P my-profile
  awstk env set -S my-stack -P my-profile

```
awstk env set [flags]
```

### Options

```
  -h, --help             help for set
  -P, --profile string   設定するプロファイル名
  -S, --stack string     設定するスタック名
```
