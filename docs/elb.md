## awstk elb cleanup

ロードバランサーを削除するコマンド

### Synopsis

指定したキーワードを含むロードバランサー（ALB/NLB/GWLB）を削除します。
削除保護が有効な場合は自動的に保護を解除してから削除します。

例:
  awstk elb cleanup -f "test-" -P my-profile
  awstk elb cleanup -f "dev" --type alb
  awstk elb cleanup -f "stg" --with-target-groups

```
awstk elb cleanup [flags]
```

### Options

```
  -f, --filter string        削除対象のフィルターパターン
  -h, --help                 help for cleanup
      --type string          ロードバランサータイプでフィルタ (alb, nlb, gwlb)
      --with-target-groups   関連するターゲットグループも削除
```

## awstk elb ls

ロードバランサー一覧を表示するコマンド

### Synopsis

ロードバランサー（ALB/NLB/GWLB）の一覧を表示します。
削除保護の状態やターゲットグループ数などの情報も含めて表示します。

【使い方】
  awstk elb ls                    # 全ロードバランサー一覧を表示
  awstk elb ls --type alb         # ALBのみを表示
  awstk elb ls --type nlb         # NLBのみを表示
  awstk elb ls --type gwlb        # GWLBのみを表示
  awstk elb ls -p                 # 削除保護が有効なもののみを表示
  awstk elb ls --details          # 詳細情報付きで表示

【例】
  awstk elb ls --type nlb -p
  → 削除保護が有効なNLBのみを一覧表示します。

```
awstk elb ls [flags]
```

### Options

```
  -d, --details          詳細情報を表示
  -h, --help             help for ls
  -p, --protected-only   削除保護が有効なもののみを表示
      --type string      ロードバランサータイプでフィルタ (alb, nlb, gwlb)
```
